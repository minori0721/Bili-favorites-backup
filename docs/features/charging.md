# 充电视频

充电专属视频不等同于普通失败，也不等同于下架。BFB在启动BBDown前检查`is_upower_exclusive`和`is_upower_play`，明确无完整权限时标记为“充电视频”。

![充电视频状态、下次权限检查时间与待清理试看残片](/screenshots/charging-video.png){.doc-shot}

## 判断规则

| 场景 | 结果 |
| --- | --- |
| 普通视频 | 正常下载 |
| 充电专属，`is_upower_play=false` | 标记充电视频 |
| 充电专属，`is_upower_play=true` | 正常下载 |
| 只有试看 | 仍视为无完整权限 |
| 字段缺失或矛盾 | 二次确认，不猜测 |
| 明确下架或不存在 | 转为已失效 |

## 检查节奏

- 全部启用账号明确无权限：`7天 ± 12小时`后复查。
- 有账号遇到风控、字段未知或临时网络错误：约`6小时 ± 30分钟`后复查。
- 没有启用账号：24小时后复查。
- 新增账号、重新扫码或启用账号：立即唤醒相关探测。

权限探测固定单并发，不占下载并发，也不显示为下载任务卡。任一账号获得完整权限后，系统使用该账号下载一次并完成所有收藏夹目标。

升级前已有的试看或无效文件不会被自动删除；它们计入“待清理残片”，不算可续传。
