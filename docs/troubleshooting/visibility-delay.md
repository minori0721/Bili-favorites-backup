# 远端延迟可见

上传完成后页面短时间显示“已上传·确认中”通常是正常状态。部分网盘在WebDAV PUT成功后不会立即通过目录列表或文件属性接口返回新文件。

## 正常行为

- BFB不重复PUT。
- 本地成品继续保留。
- 确认期间404不触发上传熔断。
- 远端同名同大小可见后自动转为最终确认。

## 何时需要处理

超过10分钟仍不可见时，任务回到待补传，并在约30分钟后重试。重试前会再次预检；如果文件已出现，会直接确认而不是重新传输。

持续失败时检查：AList目录是否正确、账号能否stat精确路径、驱动是否返回真实文件大小，以及文件是否被网盘规则改名或移动。

远端显示同名但大小不同属于冲突，不是可见性延迟，系统不会自动覆盖。
