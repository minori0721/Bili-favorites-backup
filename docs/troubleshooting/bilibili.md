# B站风控与账号失效

## 网页接口触发风控

缓存横条出现“B站触发风控，下载已暂停”时，BFB已进入固定3分钟冷却。不要频繁重启或手动反复触发任务；冷却截止时间持久化，重启不会绕过它。

冷却结束后只会放行一个探测任务。有APP token时临时用APP探测，成功取得播放地址后恢复网页模式；常驻模式不会被自动改成APP。

## 账号失效

账号卡片会显示授权刷新失败或需要重新扫码。处理步骤：

1. 在BFB重新扫码同一UID。
2. 确认账号已启用。
3. 检查APP模式下所有启用账号是否都有access token。
4. 等待相关下载、画质和充电探测任务被唤醒。

删除账号不会删除已确认备份证明。依赖该账号的运行下载会安全终止并保留会话；有其他启用账号时自动改派，没有则暂停等待同UID重新登录。

## 充电视频

显示“充电视频”不是账号故障。试看不等于完整权限，系统不会下载试看冒充备份；默认七日复查，无需手动重试。
