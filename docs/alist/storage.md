# 添加网盘与WebDAV路径

先在AList后台添加目标网盘存储，例如挂载到：

```text
/阿里云盘
```

再回到BFB全局设置，将“AList目标存储路径”填写为：

```text
/阿里云盘/bili-backup/videos
```

路径必须以`/`开头。BFB会根据目录布局继续创建账号、收藏夹或视频层级。

## 目录布局

| 设置 | 示例 |
| --- | --- |
| 用户名 / 收藏夹名 / 视频 | `/目标/账号/收藏夹/文件.mp4` |
| 收藏夹名 / 视频 | `/目标/收藏夹/文件.mp4` |
| 仅视频文件 | `/目标/文件.mp4` |

首次使用建议保留默认“用户名 / 收藏夹名 / 视频”，多账号同名收藏夹不容易碰撞。

## 连接检查

保存后先用一个小视频跑通以下动作：

1. BFB能列出目标目录。
2. PUT后进入“已上传·确认中”。
3. 稍后能读取同名文件的精确大小。
4. AList后台能下载该文件。

画质重调或旧版归档还会使用MOVE和DELETE，应在正式批量操作前单独验证。
