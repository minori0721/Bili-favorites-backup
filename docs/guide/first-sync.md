# 完成首次同步

账号添加后，选择要监控的收藏夹并保存。BFB会持续扫描热点页，同时逐步补扫历史页，不需要等整个收藏夹扫描完才开始备份。

## 第一次建议这样做

1. 保持下载并发`1`、单视频分P间隔`15秒`。
2. 先选择一个较小收藏夹，确认下载、上传与远端确认完整跑通。
3. 在队列看板观察“待下载 → 下载中 → 待上传 → 上传中”。
4. 打开AList，确认目标目录出现文件。
5. 等页面从“已上传·确认中”变为最终“已备份”。

## 三种同步动作

| 操作 | 会访问B站 | 会检查AList | 适用场景 |
| --- | --- | --- | --- |
| 立即同步 | 是 | 按正常流程 | 日常发现新增收藏 |
| 状态对账（仅AList） | 否 | 是 | 只核验远端文件 |
| 全量扫描并对账 | 是，完整扫描 | 是 | 初始化或人工强制核查 |

同步运行中再次点击会排队到下一轮，不会静默丢弃请求。

## 何时算完成

WebDAV PUT返回成功后只进入“已上传·确认中”。BFB会异步检查远端精确路径与大小；所有目标都确认后，才把关系标记为最终完成并清理本地成品。
